# Workplace Project

This repository is the project-level entrypoint for the local workspace.

## Repositories

- `workplace-api`: backend service deployed on the Huawei server
- `workplace-desktop-electron`: desktop client source

## Root-level contents

- deployment and scanning helper scripts
- project notes and evaluation reports

## Excluded local-only directories

- `work/`: generated packages and sync artifacts
- `借鉴文件夹/`: local reference material and third-party source snapshots
- `.agents/`, `.codex/`, `.pytest_cache/`: local tool state
